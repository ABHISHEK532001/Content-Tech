2. Topper Information


<p>Topper :
{
for $x in record/student
where(max(record/student/sum) = $x/sum)
        return <br>{$x/name/first_name} {$x/name/last_name} - Marks : {max(record/student/sum)}</br>
}</p>

4. Email Format Checking
                
                for $x in record/student/email
return (matches($x,'.*@.*.*'))


5. Grouping Students on Hobbies
                
                for $x in distinct-values(record/student/hobbies/text())
return(
                        for $y in record/student
                        where($y/hobbies = $x)
                        return <br>{$y/name/first_name} : {$x}</br>)


6. Ranks in Descending Order


                 for $y in sort(for $x in record/student/sum return $x)
        return  for $z in record/student 
                                where $y = $z/sum
                              return <br>Name : {$z/name/first_name}
                                               Marks : {$z/sum}</br>
         
                        
7. Students who belong to CEG Campus and have marks>200


                for $x in record/student
where(($x/sum > 200) and ($x/campus = "Anna University(CEG Campus)"))
let $name := $x/name
let $marks := $x/sum
return <br>{$name/first_name} {$name/last_name} : {$marks}</br>
